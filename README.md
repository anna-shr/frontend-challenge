# 🐱 Кошачий пинтерест

Это веб-приложение с картинками котиков и возможностью добавлять их в избранное.
Картинки загружаются из [The Cat API](https://thecatapi.com)

[Запустить приложение](https://anna-shr.github.io/frontend-challenge/)

## Использованные технологии

- **Next.js**
- **React**
- **TypeScript**
- **SCSS**
- **ESLint**
- **Autoprefixer**

### Дополнения к макету
- Счетчик избранных котиков
- Кнопка массовой очистки избранного
- Подсветка сердечек у добавленных в избранное котиков для улучшения UX
- Кружки-лоадеры при загрузке
- Компонент ошибки
- Адаптивный дизайн под разные размеры экранов
- Бесконечная прокрутка
- Кроссбраузерность
- Семантические теги
